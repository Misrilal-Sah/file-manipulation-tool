## ⚡ Quick Start

### 🐳 Option 1 — Docker (Recommended)

```bash
# Clone the repository
git clone https://github.com/your-username/pdfease.git
cd pdfease

# Launch everything in one command
docker-compose up --build
```

> 🌍 Open **http://localhost:3000** · API Docs at **http://localhost:8000/docs**

---

### 🖥️ Option 2 — Local Development

<details>
<summary><b>Frontend Setup</b></summary>

```bash
cd frontend
npm install
npm run dev
# → http://localhost:5173
```

</details>

<details>
<summary><b>Backend Setup</b></summary>

```bash
cd backend
pip install -r requirements.txt

# Optional system dependencies
# Windows: install LibreOffice + Tesseract-OCR
# Linux:   sudo apt install libreoffice tesseract-ocr

python -m uvicorn app.main:app --reload --port 8000
# → http://localhost:8000
# → Swagger UI: http://localhost:8000/docs
```

</details>

## 📡 API Endpoints

<div align="center">

| Method | Endpoint | Description |
|:------:|----------|-------------|
| `GET` | `/api/pdf/info/{file_id}` | Get PDF metadata & page count |
| `GET` | `/api/pdf/thumbnails/{file_id}` | Generate page thumbnails |
| `GET` | `/api/pdf/preview/{file_id}/{page}` | High-res page preview |
| `POST` | `/api/pdf/merge` | Merge PDFs with rotation options |
| `POST` | `/api/pdf/split` | Split PDF by page ranges |
| `POST` | `/api/pdf/rotate` | Rotate pages |
| `POST` | `/api/pdf/compress` | Compress PDF |
| `POST` | `/api/pdf/protect` | Password-protect PDF |
| `POST` | `/api/pdf/unlock` | Remove PDF password |
| `POST` | `/api/pdf/watermark` | Add watermark |
| `POST` | `/api/pdf/redact` | Redact sensitive content |
| `POST` | `/api/pdf/sign` | Add digital signature |
| `POST` | `/api/word/merge` | Merge Word documents |
| `POST` | `/api/convert/word-to-pdf` | `.docx` → `.pdf` |
| `POST` | `/api/convert/pdf-to-images` | `.pdf` → images |
| `POST` | `/api/convert/ocr` | OCR text extraction |
| `POST` | `/api/upload` | Upload file & get `file_id` |

> 📖 Full interactive API docs available at **`/docs`** (Swagger UI) and **`/redoc`** (ReDoc)

</div>

## 📁 Project Structure

```
📦 PDFease/
├── 🐳 docker-compose.yml           # Full-stack Docker orchestration
├── 🌐 render.yaml                  # Render.com deployment config

│       ├── 📂 components/
│       │   ├── FileUploader.tsx    # Drag-and-drop upload
│       │   ├── PdfPageViewer.tsx   # Page preview component
│       │   ├── RedactionPanel.tsx  # Redaction tool UI
│       │   └── SignatureSetupModal.tsx
│       ├── 📂 pages/
│       │   ├── HomePage.tsx        # Tool category landing
│       │   ├── PdfToolsPage.tsx    # All PDF operations
│       │   ├── WordToolsPage.tsx   # Word document tools
│       │   └── ConvertPage.tsx     # File conversion hub
│       ├── 📂 services/
│       │   └── api.ts              # Axios API client
│       └── 📂 store/
│           └── documentStore.ts   # Zustand global state
└── 📂 backend/
    ├── 🐳 Dockerfile
    ├── 📋 requirements.txt
    └── 📂 app/
        ├── ⚙️  config.py           # Auto-detect LibreOffice/Tesseract
        ├── 🚀 main.py              # FastAPI app + CORS + middleware
        ├── 📂 api/routes/
        │   ├── pdf.py              # PDF operation endpoints
        │   ├── word.py             # Word document endpoints
        │   ├── convert.py          # Conversion endpoints
        │   └── upload.py           # File upload endpoint
```
